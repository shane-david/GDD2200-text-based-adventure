# GDD2200-text-based-adventure
This is the final project for GDD2200. 

# Overview 
Explore the Islands of Bálfarsvegr as a Viking Leader in hopes of avenging your fallen leader Asmund. 

# Side Quest 4
For Side Quest 4 the Github repository currently contains the Unity project with every planned scene, every planned dialogue node, and an sprites folder populated with a mix of final and placeholder sprites. 

# Main Quest 2 
For Main quest 2 the Github repository currently contains the build, along with the full unity project, both are tagged and in releases for easy download. The project contains a fully function save system, text based dialouge with scriptable objects, 3 different endings, functional UI for quests, inventory, and stats and several NPCs with branching dialouge. 
- Latest Release: https://github.com/shane-david/GDD2200-text-based-adventure/releases/tag/v1.0.0

# Ending Guide
There are three endings in the game:
- Óbrjótanligr vilji "Unbreakable Will" - obtained by beating the game having, burned your boats and having a humanity score greater than or equal to 10 
- Hetja Fórn "A Hero's Sacrifice" - obtained by beating the game, having burned your boats, and having a humanity score less than 10
- Skamfararferð "Shameful Return" - obtained by beating the game and having not burned your boats.
